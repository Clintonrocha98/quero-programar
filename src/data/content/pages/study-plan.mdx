      ## FASE 3 — BANCO DE DADOS RELACIONAL & SQL

      **Objetivo:** Pensar sistemas a partir de dados.

      ### Conteúdos
      - Conceitos fundamentais: tabelas, linhas, colunas, relações
      - Modelagem relacional: 1:1, 1:N, N:N
      - Chaves: primária, estrangeira
      - SQL básico: <b>SELECT</b>, <b>INSERT</b>, <b>UPDATE</b>, <b>DELETE</b>
      - SQL intermediário: <b>WHERE</b>, <b>ORDER BY</b>, <b>LIMIT</b>, <b>JOIN</b>, <b>GROUP BY</b>
      - Diferença prática entre MySQL e PostgreSQL

      <CodeBlock 
        filename="queries.sql"
        language="sql"
        code={`-- Criar tabela de usuarios
CREATE TABLE users (
    id SERIAL PRIMARY KEY,
    name VARCHAR(100),
    email VARCHAR(100) UNIQUE
);

-- Buscar usuarios ativos
SELECT * FROM users WHERE active = true;

-- Relacionar usuarios e pedidos
SELECT users.name, orders.total 
FROM users 
JOIN orders ON users.id = orders.user_id;`}
      />

      ### Entregáveis
